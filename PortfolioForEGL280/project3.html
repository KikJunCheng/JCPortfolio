Project Summary

Overview

This project is a Python-based music visualisation created for the song “Light It Up”. The visualisation transforms individual pixels into a dynamic graphical representation of the music, synchronising visual elements with the rhythm and intensity of the song. By analysing the audio, the program animates pixels frame by frame based on the beat, creating an immersive and responsive visual experience.

Technical Approach

The project uses Python to process audio data and convert it into visual patterns that react in real time to the music. Pixel-based graphics are animated sequentially, with timing and motion driven by beat detection and amplitude changes. This approach required precise control over frame updates, timing synchronisation, and performance optimisation to ensure smooth animation throughout the song.

Challenges

One of the main challenges in developing this game was managing the The biggest challenge occurred three days before the submission deadline, when a critical blue screen failure on the laptop resulted in the loss of the project. This required restarting the entire development process and rewriting over 3,000 lines of code within a very limited timeframe. While this setback was stressful, rebuilding the project from scratch allowed for cleaner logic, improved structure, and better optimisation. As a result, the final visualisation was significantly stronger and was ultimately selected for the end-of-semester showcase, turning a major technical failure into a key learning experience.
