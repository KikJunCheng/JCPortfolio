Project Summary

Overview

This project is a full-stack programming workshop website developed using HTML, CSS, JavaScript, PHP, and SQL. The platform is designed to allow users to browse, purchase, and manage programming workshops through a complete user account and order management system. The focus of this project is on usability, real-world functionality, and database-driven interactions.

Core Mechanics

The workshop website provides a complete user experience through a full account management system, allowing users to register, log in, edit their profiles, and reset forgotten passwords, with personal details and accumulated points displayed on the profile page. A points-based rewards system enables users to redeem a 20% discount voucher upon reaching 500 points, which is automatically validated during checkout. Users can browse and book workshops by selecting their preferred timing and quantity, supported by a floating cart interface for quick access and improved usability. The platform also includes a comprehensive checkout and order management system, where users can apply vouchers, view order history, amend orders within five days, or delete them if necessary. Additionally, a search function helps users quickly locate workshops, while a feedback system allows users to submit star ratings and comments that are displayed on the homepage to enhance transparency and user trust

Ensuring that data was correctly stored, retrieved, and updated across multiple tables required careful database design and debugging. Handling backend processes such as order creation, profile updates, and voucher eligibility checks was significantly more complex than frontend implementation.

Another challenge involved managing workshop scheduling logic, specifically handling the selection between Timing A and Timing B using JavaScript and ensuring the correct data was passed and processed on the server side. Due to time constraints, the implementation of the points and voucher system was completed under a tight schedule, which required prioritizing core functionality over additional optimizations. This experience highlighted the importance of backend planning, efficient time management, and incremental feature development.
